暴力破解zip

2022-03-16
python自动化办公

作为老书虫，最喜欢就是看书了，偶然得到很多专业课的书，居然还得要解压码，含泪出了一元买了其中的一个，越想越气怎么可以出线呢？

程序猿上才艺，暴力解决

暴力解耗费时间长的很的，排列组合算一下，如果按照6位数，差不多也得上亿情况，我这个是刚好看到，他图片给的压缩码的前两位和后一位，而且我买了其中一个解压码，推测解压码有6位，所以只需要迭代中间三位
仅供私下自己使用，千万别传播啊~，要是老板知道他的解压码，被咱这样玩出来，以后没发白嫖了，滑稽.jpg, 狗头.jpg

做完这个我也有点儿其他想法，要是zip可以分片，当我获得到正确密码后，不再进行解压，讲密码输出，存放在对应目录就好了，而且对于大文件zip，解压次数多了以后，时间等的有点儿长

import zipfile
from threading import Thread


# TODO 破解****下载的 it书籍的zip
def passwd(zFile, pwd, jieya_path):
    # print(f'正在尝试密码: {pwd}')
    try:
        zFile.extractall(jieya_path, pwd=str(pwd).encode('utf-8'))
        print(f'解压成功,密码是:{pwd}')
        with open(jieya_path + pwd + ".txt", "w") as f:
            f.write(pwd)
        zFile.close()
    except Exception as e:
        pass


def create_pwd(len):
    import itertools as its
    words = '1234567890QWERTYUIOPASDFGHJKLZXCVBNMqwertyuiopasdfghjklzxcvbnm'
    base = its.product(words, repeat=len)
    for i in base:
        yield ''.join(i)


if __name__ == "__main__":
    # passwd('./base_data/aa.zip')
    # passwd 自己手动写入猜写密码
    jieya_path = "D:\Download\IT书籍\《零起点Python机器学习快速入门》_何海群\\"
    zFile = zipfile.ZipFile(jieya_path + "a.zip")

    # 传入未知密码个数 2位或者3位
    for p in create_pwd(2):
        # 已知其中的前2位+未知+后两位
        pwd = "e3" + p + "k7"
        # 多线程破解
        t = Thread(target=passwd, args=(zFile, pwd, jieya_path))
        t.start()
