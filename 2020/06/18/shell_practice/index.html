练手shell题目

2020-06-18
Linuxshell

出一些shell题目练练手

1. 用while循环输出1-100的和
#!/bin/bash
i=1
sum=0
while(( $i <= 100))  #!版本问题 -le代替<=
do
       let sum=sum+$i
       let i++
done

echo $sum
2. for循环输出1-100
#!/bin/bash
sum=0
for (( i=1;i <= 100;i=i+1 ))
do 
	let sum=sum+i
done
echo "（for）从1到100的和为：$sum"
3. 计算输入的参数1和参数2的两个数的和，并输出到控制台
#!/bin/bash   
function sum()    
{   
    s=0;    
    s=$[$1+$2]    
echo “$s”
}  
read -p "Please input the number1: " n1;   
read -p "Please input the number2: " n2;   
sum $n1 $n2;
4. until循环输出1-100的和
#!/bin/bash
i=0
sum=0
until ((i>100))
do
		sum=`expr $sum + $i`
		i=`expr $i + 1`
done

echo "1+2+3+……+100= $sum"
5. while循环输出0-100
#!/bin/bash
i=0
while((i<=100))
do
	echo “The number is $i”
	let i=$i+1
done
6. 用for循环输出当前文件夹下面的所有文件
#!/bin/bash
for i in $(ls)
do
    echo item:$i
done
7. 用for循环输出当前文件夹下面的所有文件,统计文件数量
#!/bin/bash
num=0
for i in $(ls)
do
    num=$[$num+1]
    echo item $num: $i
done

echo "The number of files is $num"
8. 用两种方法运行shell的脚本文件/home/test/test.sh
第一种：输入脚本的绝对路径或相对路径
（1）首先要赋予test.sh脚本的+x权限
# cd /home/test
# chmod 777 test.sh
（2）执行脚本
# ./test.sh

第二种：bash或sh+脚本（不用赋予脚本+x权限）
# sh /home/test/test.sh
9. shell脚本获取本机的网络地址
#!/bin/bash
# 具体file路径看Linux版本
file="/etc/sysconfig/network-scripts/ifcfg-eth0"
if [ -f $file ]
	then
        IP=`grep "IPADDR" $file|awk -F"=" '{ print $2 }'`
        MASK=`grep "NETMASK" $file|awk -F"=" '{ print $2 }'`
		echo "$IP/$MASK" 
		exit 1 #异常报错
fi
10. 判断输入的用户名是否为root和test

如果是”hello “，如果都不是提示”sorry” 

#!/bin/bash
read -p "Please input your name:" NAME
printf "%s\n" $NAME

if [[ $NAME = root ]] || [[ $NAME = test ]]
then
	echo “hello ${NAME},welcome !”
else 
	echo “sorry”
fi
11. 添加一个新组为class1，

然后添加属于这个组的30个用户，用户名的形式为stdxx，其中xx从01到30

#!/bin/bash
groupadd class1
for ((i=1;i<=30;i++))
do
    if [ $i -le 9 ];then
    	USERNAME=std0${i}
    else
    	USERNAME=std${i}
    fi
    useradd -G class1 $USERNAME
done
12. 实现自动删除30个账号的功能。账号名为std01至std30。
#!/bin/bash
for((i=1;i<=30;i++))
do
    if [ $i -lt 10 ];then
    	username="std0"$i
    else
    	username="std"$i
    fi
    userdel -r $username
done

14. 判断一文件是不是字符设备文件，

如果是将其拷贝到 /dev 目录下。要求必须有提示信息：Input file name

#!/bin/bash
FILENAME=echo "Input file name:"
read FILENAME
if [ -c "$FILENAME" ]
then
		cp $FILENAME /dev
else
		echo "不符合要求"
fi
